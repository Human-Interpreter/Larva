# Larva Game Project
마피아에 카드요소를 이용하는 게임 제작 프로젝트 입니다.

## 개발 일정
- 기능 개발 (9월, 10월)
- 게임 퀄리티 업, 디버깅, 게임 밸런싱 (11월)
- 스토어 배포 (12월)
